# DAGPENGER-METRICS
Et lite bibliotek som tilbyr [prometheus-metrikker](https://prometheus.io/docs/concepts/metric_types/) med dagpenger-spesifikk konfigurasjon lagt på.

# Komme i gang

Gradle brukes som byggverktøy og er bundlet inn.

`./gradlew build`
